Missing Node.js

We couldn't find Node.js on your machine. If you've installed Node.js, make sure node is in your PATH. If you haven't installed Node.js yet, you can get it here.

If you've installed Node.js already, it may be missing from your PATH. Here's how to add it:

  1. Open the Start Menu and search for "Environment Variables".
  2. Click "Edit System Environment Variables" and then the "Environment Variables" button.
  3. Look for the "Path" variable under "System Variables", and double-click it.
  4. Click "New", and enter C:\Program Files\nodejs, or wherever your Node.js installation is located at.
  5. Log out and in again, or reboot your machine.

Alternately, if you use Homebrew, you can run brew install node from the command line.

The version of Node.js you have installed is fairly old, we recommend upgrading to avoid running into errors with code written against an older version. You can download the latest version of Node.js here.

Alternately, if you installed Node.js via Homebrew, you can run brew upgrade node from the command line.
